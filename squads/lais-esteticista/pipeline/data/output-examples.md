## Exemplo 3: Carrossel — Educativo com Desmistificação

**Referência**: Padrão consolidado dos 3 perfis
**Formato**: Carrossel (6 slides)
**Tema**: Limpeza de pele — o que realmente é

### Slides
```
SLIDE 1 (Hook):
Você está fazendo limpeza de pele do jeito certo?

SLIDE 2:
❌ Mito: limpeza de pele é só para quem tem acne

SLIDE 3:
✅ Verdade: toda pele se beneficia
→ Controla oleosidade
→ Melhora textura
→ Previne envelhecimento precoce

SLIDE 4:
⏱️ Com que frequência?
Depende da sua pele.
Uma avaliação profissional define a frequência ideal.

SLIDE 5:
🌿 O que esperar do procedimento:
→ Higienização profunda
→ Extração segura
→ Hidratação finalizada

SLIDE 6 (CTA):
Sua pele merece cuidado profissional.
Agende sua avaliação.
@lais_esteticista
```

### Caption
```
A limpeza de pele não é só para quem tem acne ✨

Toda pele — oleosa, seca, mista ou sensível — se beneficia de uma limpeza profissional regular.

A frequência ideal? Só uma avaliação profissional pode te dizer, porque cada pele tem suas próprias necessidades.

Salva esse post para ter sempre à mão 💛
@lais_esteticista

#esteticafacial #limpezadepele #cuidadoscomapele #skincare #estetica
```

**Score estimado**: 9/10
**Por que funciona**: Desmistifica (slide 1), educa (slides 2-5), fecha com CTA suave. Caption pede o save.

## Exemplo 5: Stories — Sequência de Bastidores

**Referência**: Padrão de @bettinaesteticista e @lais_esteticista
**Formato**: Stories (3 stories em sequência)
**Tema**: Bastidores de atendimento

### Story 1
```
Visual: clipe rápido da cabine/espaço preparado
Texto sobrepost: "Preparando tudo para receber você ✨"
```

### Story 2
```
Visual: close em produto ou equipamento
Texto: "Você sabe para que serve isso? 👇 [caixa de pergunta]"
```

### Story 3
```
Visual: resultado final ou produto usado
Texto: "Sua pele merece esse cuidado. Agenda aberta!"
CTA: @lais_esteticista | [link/botão]
```

**Score estimado**: 8/10
**Por que funciona**: Bastidor cria conexão, caixa de perguntas gera interação, CTA no momento certo.
