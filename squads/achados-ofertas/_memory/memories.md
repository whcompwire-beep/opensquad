## Investigação Sherlock (2026-03-26)

### Perfis analisados
- **@oreidapromobr** — 1,5M seguidores: usa códigos proprietários (OREIBR-XXXX, GZRAMO-XXXX) para ML, legendas curtas, CTA via comentário do código
- **@achadosoriginais** — 1,7M seguidores (verificado): foco em moda/tênis, keyword-CTA gera 300+ comentários (ex: DUNK→320 comentários), formato "Revew |" para reviews
- **@brasilpromoofertas** — 1,3M seguidores: agregador Shopee 100% template-driven, catálogo numerado (200+ produtos), anti-shadowban com ortografia alternativa ("S!ga")

### Padrões universais identificados
1. **Reels dominam** — 70-80% do conteúdo, maior alcance orgânico
2. **Keyword-CTA é universal** — "Comenta [PALAVRA] que mando" em 100% dos posts de Feed/Reels
3. **WhatsApp group CTA** — presente em 90%+ dos posts como segundo CTA
4. **Sem URLs na legenda** — todos usam comentários, DMs ou Stories para links
5. **Preço explícito** — sempre no corpo da legenda, não escondido

### Framework adotado
- Keyword-CTA temática (não genérica) por produto
- Código RIOMAFRA-CATEGORIA-NNN para produtos ML
- Pipeline Garimpeiro → Estrategista → Copywriter → Revisor
- 3 checkpoints de aprovação humana: produtos, estratégia, conteúdo

## Aprendizados de Execução

### Run 2026-03-26-120000 (primeira execução)
- **Média de qualidade:** 94,7/100 — todos os 3 produtos aprovados
- **Preferência do William:** Feed = imagem única (não carrossel)
- **Melhor hook da rodada:** Produto Pet — "Esse pet não quer mais sair da cama e honestamente entendo ele 😭" (30/30)
- **Observações para próximas rodadas:**
  1. Sempre incluir percentual de desconto explícito (50% OFF) nas legendas
  2. Validação pessoal ("Comprei aqui, aprovei.") deve estar no Reels, não só no WhatsApp
  3. Preço original → preço atual cria contraste visual forte (ex: "R$119 → R$59")
  4. Variante B do Pet ("o pet dormiu, a gente viu o prejuízo 😂") tem alto potencial viral — testar
