## Templates por Plataforma (para o Copywriter)

### Instagram Reels — Template padrão
```
[HOOK — máx 125 chars, sem nome do produto]
[BENEFÍCIO ou CONTEXTO — 1-2 linhas]
[PREÇO + DESCONTO + AVALIAÇÃO — 1 linha]
Comenta [KEYWORD] que mando o link! 👇
Ainda não está no grupo? Comenta GRUPO!
#[hashtag1] #[hashtag2] #[hashtag3] (máx 5)
```

### Instagram Feed (Carrossel) — Slides
```
SLIDE 1: [Frase de impacto] + emoji visual
SLIDE 2: [Produto + principal benefício] + foto produto
SLIDE 3: [Preço, desconto, avaliações] + números em destaque
SLIDE 4: [Como comprar — código ou instrução de 1 linha]
SLIDE 5: [CTA + keyword] + "Ainda não está no grupo?"

LEGENDA:
[Hook — aparece antes do "ver mais"]
[1-2 linhas de contexto]
Comenta [KEYWORD] que mando o link! 👇
Ainda não está no grupo? Comenta GRUPO!
[máx 5 hashtags]
```

### Instagram Stories — Template urgência
```
🔥 [PRODUTO] — R$[X] HOJE
[Flash sale / Termina em Xh]
Comenta [KEYWORD] ↙️ ou swipe up
```

### WhatsApp Broadcast — Template padrão
```
Oi! 👋

[ACHADO DO DIA — 1 frase de hook]

[PRODUTO]: R$[X] [na Shopee/ML] — [desconto]% OFF
Avaliação: [X.X] ⭐ | [N.NNN] compradores
[Frete grátis / Entrega em X dias]

[Instrução de compra — 1 linha]
Comenta QUERO que mando o link direto 👇

— Achados RioMafra 👑
```
