# Domain Framework — Achados RioMafra

## O Domínio: Marketing de Afiliados em Redes Sociais

### Modelo de Negócio
- **Fonte de receita:** Comissões por venda através de links de afiliado (Mercado Livre, Shopee, Amazon)
- **Produto:** Atenção do seguidor convertida em clique → compra
- **Diferencial competitivo:** Curadoria confiável + descoberta antes dos outros afiliados

### A Jornada do Seguidor
```
Scroll no feed
    ↓
Hook para o scroll (< 2 segundos)
    ↓
Lê a legenda / assiste o Reel (10-30 segundos)
    ↓
Comenta a keyword / vai para os Stories
    ↓
Recebe o link / clica
    ↓
Compra → Comissão 💰
```

## Estrutura de Conteúdo

### Fórmula Universal de Alta Performance
Baseada na análise de @oreidapromobr (1,5M), @achadosoriginais (1,7M), @brasilpromoofertas (1,3M):

```
HOOK (para o scroll) → CONTEXTO (por que importa) → DADOS (preço/avaliação) → CTA (comentário/link)
```

### Fórmulas de Hook Testadas

**Reação + Produto:**
> "[Produto] me surpreendeu demais! [Veredicto breve]."

**Transformação:**
> "Minha [situação ruim]… esse [produto] por R$[X] [resolveu]."

**Urgência:**
> "[PRODUTO] NA PROMO ‼️ Por R$[X] [usando cupom X]."

**Curiosidade/Subversão:**
> "Tirem as crianças da sala — produto de adulto aqui 🔞 [produto inocente]"

**Descoberta:**
> "Achei [na Shopee/no ML] e não consigo guardar pra mim 👀"

**Comparação:**
> "Eu tava pagando R$[X]... esse aqui custa R$[Y] e faz a mesma coisa"

**Presente:**
> "Seria um ótimo presente 👀 [termo de busca/código]"

## Sistema de Keyword-CTA

O keyword-CTA é o mecanismo central de engajamento para Feed e Reels:

```
Comenta [KEYWORD] que mando o link! 👇
```

**Por que funciona:**
1. Aumenta engajamento (comentários) → sinaliza relevância ao algoritmo
2. Cria lista de interessados qualificados
3. Evita shadowban (não expõe link direto na legenda)
4. Facilita respostas automáticas via DM

**Regras para a keyword:**
- Temática para o produto (ORGANIZAR, DUNK, QUERO, PET...)
- 1 palavra, maiúscula, fácil de digitar
- Evitar palavras genéricas (LINK, QUERO, PROMO) — preferir temáticas

---

## Sistema de Códigos Anti-Shadowban (Mercado Livre)

Para produtos do Mercado Livre, o sistema usa códigos de busca internos:

**Formato:** `RIOMAFRA-[CATEGORIA]-[NÚMERO]`

**Categorias:**
- `CASA` — Organização, decoração, cozinha
- `TECH` — Eletrônicos, acessórios de celular
- `MODA` — Roupas, calçados, acessórios
- `BELEZA` — Cosméticos, cuidados pessoais
- `PET` — Produtos para animais
- `SPORT` — Esporte e lazer
- `KIDS` — Infantil

**Uso:** O seguidor digita o código no ML e encontra o produto. O afiliado posta o código nos comentários (não na legenda).

## Validação de Produto (Critérios mínimos)

Um produto SÓ entra na pipeline se atender:
- ⭐ Mínimo 4.0 estrelas
- 💬 Mínimo 100 avaliações
- 💰 Desconto real verificado (não "de por" inflado)
- 🚚 Frete grátis (preferencial)
- 🔗 Programa de afiliado ativo
- 📦 Estoque disponível
