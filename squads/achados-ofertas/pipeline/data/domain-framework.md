# Domain Framework — Achados RioMafra

## O Domínio: Marketing de Afiliados em Redes Sociais

### Modelo de Negócio
- **Fonte de receita:** Comissões por venda através de links de afiliado (Mercado Livre, Shopee, Amazon)
- **Produto:** Atenção do seguidor convertida em clique → compra
- **Diferencial competitivo:** Curadoria confiável + descoberta antes dos outros afiliados

### A Jornada do Seguidor
```
Scroll no feed / TikTok / Facebook
    ↓
Hook para o scroll (< 2 segundos)
    ↓
Lê a legenda / assiste o Reel (10-30 segundos)
    ↓
Comenta a keyword / vai para os Stories / clica no link (Facebook/TikTok bio)
    ↓
Recebe o link / clica
    ↓
Compra → Comissão 💰
```

## Mercado Local — Fase 1: Mafra-SC e Rio Negro-PR

### Contexto do mercado
- **Mafra-SC:** ~45.000 habitantes, planalto norte catarinense
- **Rio Negro-PR:** ~42.000 habitantes — cidades conurbadas (mesma área urbana)
- **Perfil do público:** Famílias de classe média, trabalhadores, pequenos comerciantes, donas de casa
- **Renda estimada:** R$3.000–R$10.000/mês por família (foco em R$4k–7k)

### Estratégia de crescimento
1. **Fase 1 (atual):** Crescimento orgânico + impulsionamento geolocalizado (Meta Ads: Mafra + Rio Negro)
2. **Fase 2:** Escala regional — expandir raio para planalto norte/sul SC e PR
3. **Fase 3:** Nacional — afiliados em escala sem segmentação geográfica

### Impulsionamento com Meta Ads (Fase 1)
- Segmentação: Mafra, SC + Rio Negro, PR + raio de 30 km
- Público: 22–55 anos, interesses em economia doméstica, promoções, compras online
- Formato recomendado para impulsionar: Reels (maior alcance) + Stories (link direto)
- Budget inicial: R$10–30/dia por post impulsionado

## Estratégia de Faixa de Preço — Classe Média

### Sweet spot de conversão

| Faixa de Preço | Classificação | Exemplos de categoria | Prioridade |
|---|---|---|---|
| R$15–R$80 | Compra por impulso | Organização, beleza, acessórios, pet, cozinha | 🔥 Alta |
| R$80–R$250 | Compra considerada | Tech, moda, cama/banho, eletro pequeno | ✅ Média |
| R$250–R$500 | Compra planejada | Eletrônicos maiores, kit presentes | 📌 Baixa |
| +R$500 | Evitar (Fase 1) | Smartphones, TVs, móveis | ❌ Não incluir |

### Regra do desconto mínimo
- Compra por impulso (R$15–R$80): mínimo **30% de desconto** ou preço absurdamente baixo
- Compra considerada (R$80–R$250): mínimo **20% de desconto** ou cupom que muda o cálculo
- Sempre exibir "de R$X por R$Y" — o contraste visual é o gatilho de conversão

### Categorias de alta prioridade para este mercado
**Impulso (alta conversão, alta frequência):**
- Organização de gavetas, prateleiras e cozinha
- Produtos de beleza e cabelo
- Acessórios para pet
- Utensílios de cozinha
- Acessórios para celular e carregadores

**Considerados (bom ticket, boa comissão):**
- Roupas e calçados em promoção real (não apenas "de por" inflado)
- Pequenos eletrodomésticos (fritadeira, liquidificador, etc.)
- Produtos para bebê e criança

## Estrutura de Conteúdo

### Fórmula Universal de Alta Performance
Baseada na análise de @oreidapromobr (1,5M), @achadosoriginais (1,7M), @brasilpromoofertas (1,3M):

```
HOOK (para o scroll) → CONTEXTO (por que importa) → DADOS (preço/avaliação) → CTA (comentário/link)
```

### Fórmulas de Hook Testadas

**Reação + Produto:**
> "[Produto] me surpreendeu demais! [Veredicto breve]."

**Transformação:**
> "Minha [situação ruim]… esse [produto] por R$[X] [resolveu]."

**Urgência:**
> "[PRODUTO] NA PROMO ‼️ Por R$[X] [usando cupom X]."

**Curiosidade/Subversão:**
> "Tirem as crianças da sala — produto de adulto aqui 🔞 [produto doméstico inocente]"

**Descoberta:**
> "Achei [na Shopee/no ML] e não consigo guardar pra mim 👀"

**Comparação:**
> "Eu tava pagando R$[X]... esse aqui custa R$[Y] e faz a mesma coisa"

**Presente:**
> "Seria um ótimo presente 👀 [termo de busca/código]"

## Plataformas e Seus Papéis

| Plataforma | Papel | Horário ideal | Gatilho principal | CTA |
|---|---|---|---|---|
| **Reels (Instagram)** | Descoberta / Algoritmo | 19h–21h | Curiosidade / Entretenimento | Keyword-CTA nos comentários |
| **Feed (Carrossel)** | Arquivo / Saves | 12h–14h | Informação / Comparação | Keyword-CTA nos comentários |
| **Stories (Instagram)** | Urgência / Link direto | 11h–12h, 18h–19h | Urgência / FOMO | Swipe up / Caixa de resposta |
| **WhatsApp** | Conversão direta | 12h, 20h | Proximidade / Confiança | "Comenta QUERO" ou link direto |
| **Facebook** | Descoberta + Conversão | 12h–14h, 19h–21h | Review / Custo-benefício | Link nos comentários do post |
| **TikTok** | Descoberta / Viral | 18h–22h | Entretenimento / Surpresa | Link na bio |

### Notas sobre Facebook (interior / classe média 35+)
- Facebook é o canal dominante para faixa 35–60 anos em cidades do interior
- Posts com vídeo (Reels compartilhados) têm o melhor alcance orgânico
- Links podem ser colocados diretamente nos comentários do post (sem shadowban)
- Grupos locais de "Compra e Venda" e "Achados" são canais de distribuição orgânica poderosos
- Caption pode ser mais longa que no Instagram (o público lê mais)

### Notas sobre TikTok (18–35 anos, crescendo no interior)
- Hook visual nos primeiros 3 segundos decide se o vídeo vai ou não
- Caption curtíssima — máximo 2 linhas + hashtags
- Link só na bio — nunca na legenda
- Conversão acontece via link na bio → WhatsApp/grupo de ofertas
- Música/áudio trending aumenta alcance orgânico significativamente

## Sistema de Keyword-CTA

O keyword-CTA é o mecanismo central de engajamento para Feed e Reels do Instagram:

```
Comenta [KEYWORD] que mando o link! 👇
```

**Por que funciona:**
1. Aumenta engajamento (comentários) → sinaliza relevância ao algoritmo
2. Cria lista de interessados qualificados
3. Evita shadowban (não expõe link direto na legenda)
4. Facilita respostas automáticas via DM

**Regras para a keyword:**
- Temática para o produto (ORGANIZAR, DUNK, QUERO, PET...)
- 1 palavra, maiúscula, fácil de digitar
- Evitar palavras genéricas (LINK, QUERO, PROMO) — preferir temáticas

## Sistema de Códigos Anti-Shadowban (Mercado Livre)

Para produtos do Mercado Livre, o sistema usa códigos de busca internos:

**Formato:** `RIOMAFRA-[CATEGORIA]-[NÚMERO]`

**Categorias:**
- `CASA` — Organização, decoração, cozinha
- `TECH` — Eletrônicos, acessórios de celular
- `MODA` — Roupas, calçados, acessórios
- `BELEZA` — Cosméticos, cuidados pessoais
- `PET` — Produtos para animais
- `SPORT` — Esporte e lazer
- `KIDS` — Infantil

**Uso:** O seguidor digita o código no ML e encontra o produto. O afiliado posta o código nos comentários (não na legenda).

## Validação de Produto (Critérios mínimos)

Um produto SÓ entra na pipeline se atender:
- ⭐ Mínimo 4.0 estrelas
- 💬 Mínimo 100 avaliações
- 💰 Desconto real verificado (não "de por" inflado)
- 💵 Preço dentro do sweet spot da classe média (idealmente R$15–R$250)
- 🚚 Frete grátis (preferencial)
- 🔗 Programa de afiliado ativo
- 📦 Estoque disponível
