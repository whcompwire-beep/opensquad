# Step 03 — Estratégia de Conteúdo

## Metadata
- **Agent:** Estrategista
- **Input:** Relatório de produtos do Garimpeiro + brief visual
- **Output:** Mapa de conteúdo completo com ângulos, keywords, formatos e briefing para o Copywriter
- **Tempo esperado:** 5–10 minutos

---

## Objetivo

O Estrategista seleciona os melhores produtos do relatório do Garimpeiro, define o ângulo de conteúdo ideal para cada um, atribui keyword-CTAs para Instagram e estratégias de link para todas as plataformas, e cria um mapa de publicação com horários.

---

## Instruções de Execução

### 1. Ler o Relatório do Garimpeiro
Para cada produto:
1. Qual é o ângulo mais forte?
   - `curiosidade` — produto surpreende, tem "antes e depois"
   - `urgência` — flash deal, estoque limitado, prazo real
   - `review` — prova social forte (4.8+ stars, 1000+ avaliações)
   - `comparação` — preço muito menor que a concorrência
   - `presente` — item perfeito para presente em data especial
   - `rotina` — produto que resolve problema do cotidiano
2. Qual plataforma vai gerar mais resultado?
3. Qual keyword temática para Instagram?
4. O produto tem apelo visual para Reel/TikTok?
5. Potencial de compartilhamento em grupos locais no Facebook (Mafra/Rio Negro)?

### 2. Selecionar e Priorizar
**Máximo 3 produtos** por rodada:
- 1 produto 🔥 **URGENTE** → publicação imediata
- 1–2 produtos ✅ **RECOMENDADOS** → publicação programada

### 3. Criar o Mapa de Conteúdo
Para cada produto, definir:
- Ângulo principal e justificativa
- Keyword-CTA para Instagram (1 palavra, maiúscula, temática)
- Estratégia de link por plataforma:
  - **Instagram Feed/Reels:** keyword-CTA nos comentários → DM automática
  - **Instagram Stories:** link direto (swipe-up ou caixinha de resposta)
  - **Facebook:** link do afiliado nos primeiros comentários do post
  - **TikTok:** link na bio → WhatsApp/grupo
  - **WhatsApp:** link direto na mensagem
- Quais plataformas e formatos cobrir
- Ordem de publicação com horários específicos
- Chamadas especiais (evento, flash deal, etc.)

### 4. Variantes A/B para o Produto Principal
- **Hook A:** Curiosidade/subversão
- **Hook B:** Urgência/preço direto

### 5. Briefing para o Copywriter
O briefing deve deixar o Copywriter sem nenhuma decisão estratégica pendente:
- Plataformas, ângulo, keyword (Instagram) e estratégia de link (todas as plataformas)
- Tom de voz desta rodada
- Variantes A/B
- Instruções especiais (código RIOMAFRA, menção de prazo de entrega, etc.)

---

## Horários Recomendados por Plataforma

| Plataforma | Horário ideal | Justificativa |
|---|---|---|
| Stories | 11h–12h | Antes do almoço — flash deals urgentes |
| WhatsApp | 12h, 20h | Pico de abertura de mensagens |
| Facebook | 12h–14h, 19h–21h | Público 35+ ativo no almoço e noite |
| Reels | 19h–21h | Pico de audiência Instagram |
| Feed | 12h–14h | Horário de almoço — mais saves |
| TikTok | 18h–22h | Pico TikTok, especialmente 20h–22h |
