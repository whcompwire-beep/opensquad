# Checkpoint — Revisão do Copy

O Copywriter criou o copy completo de todos os posts. Revise antes de criar as imagens.

---

**O que revisar:**
- Texto de cada slide do carrossel (headline + corpo)
- Texto da imagem única
- Texto dos frames dos Stories + elemento interativo
- Legendas completas e hashtags de cada post

**Você pode:**
- Aprovar e seguir para a criação das imagens
- Ajustar o texto de algum slide específico
- Mudar algum CTA ou hashtag
- Pedir reescrita de algum elemento

**Responda:**
- O copy está aprovado? Ou há ajustes a fazer antes de criar as imagens?
