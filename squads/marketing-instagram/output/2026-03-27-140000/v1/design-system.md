# Design System — @lais_esteticista — Inverno 2026

## Platform Specs
- Carrossel Feed: 1080 x 1440px (3:4 portrait)
- Imagem Única Feed: 1080 x 1080px (1:1 square)
- Stories: 1080 x 1920px (9:16 vertical)

## Colors
| Token | Hex | Uso |
|-------|-----|-----|
| Primary | #2D2D2D | Headlines e texto em fundos claros |
| Background Light | #F5EDE3 | Fundo slides claros — bege warm |
| Accent | #C97D6E | Terracota rosado — destaques, accent words, CTA |
| Background Dark | #3D2B26 | Fundo slides escuros — marrom aconchegante |
| Text Light | #FAF6F2 | Texto em fundos escuros — branco quente |
| Muted | #A89080 | Texto secundário, captions, elementos suaves |

## Typography
| Role | Family | Size (Carousel) | Size (Stories) | Weight |
|------|--------|-----------------|----------------|--------|
| Hero | Playfair Display | 62px | 58px | 700 |
| Heading | Playfair Display | 48px | 48px | 700 |
| Body | Inter | 36px | 38px | 400 |
| Caption | Inter | 26px | 28px | 400 |

## Spacing
- Base unit: 24px
- Content margin: 80px (carrossel), 70px (stories)
- Section gap: 48px
- Between headline and body: 32px

## Visual Elements
- Border radius: 0 (clean, editorial)
- Accent highlight: span com cor #C97D6E inline no headline
- Brand mark: nome "@lais_esteticista" em caption, topo direito, cor Muted
- Decorative: linha horizontal 2px accent (#C97D6E) abaixo do headline quando relevante

## Notes
- Playfair Display importada via Google Fonts @import
- Inter importada via Google Fonts @import
- Todos os HTMLs são self-contained
- Nunca usar numeração de slides (ex: "1/8")
